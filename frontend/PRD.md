# Lerumos Political Party

A comprehensive social media platform for political discourse where users can engage with content based on their political affiliation, earn rewards through community engagement, promote themselves through premium advertising features, and participate in a fully-featured political community with robust administrative controls.

**Experience Qualities**: 
1. **Engaging** - Interactive political discussions with gamified reward system and tipping mechanics
2. **Partisan** - Clear visual distinctions between political affiliations with themed UI elements
3. **Community-driven** - Rich social features including follow/unfollow, messaging, and profile customization
4. **Professional** - Enterprise-grade admin controls, settings management, and support systems

**Complexity Level**: Complex Application (advanced functionality with multiple views and administrative features)
The platform now includes comprehensive user management, monetization systems, administrative controls, support systems, trending topics, calendar-based ad placement, certificate generation, token redemption, username marketplace, profile ad slots, and custom API endpoint management.

## Essential Features

**Political Affiliation Selection**
- Functionality: Users choose Democrat, Republican, or Independent alignment
- Purpose: Customizes UI theme and content algorithm preferences  
- Trigger: During signup or profile editing
- Progression: Select affiliation → UI updates with party colors → personalized feed appears
- Success criteria: Profile displays correct party colors and themed content

- Purpose: Core social engagement and content generation
- Trigger: Click "Post" button or navigate to Meme Wall
- Progression: Compose content → add media → select visibility → publish → appears in feeds
- Success criteria: Posts appear in appropriate party feeds and can receive interactions

**Engagement & Tipping System**
- Functionality: Like/dislike posts using party symbols, tip high-performing content creators
- Purpose: Gamify political engagement and reward quality content
- Trigger: Interact with posts or view daily tips dashboard
- Progression: React to posts → earn engagement points → qualify for tips → cash out earnings
- Success criteria: Users receive tips based on engagement, platform takes 10% fee

- Functionality: Friend requests, following system, private messaging
- Purpose: Build political communities and enable direct communication

- Progression: Send friend request → acceptance → access to private content → direct messaging

**Meme Wall**

## Edge Case Handling
- **Missing party affiliation**: Default to Independent with neutral colors
- **Inappropriate content**: Basic content flagging system with community moderation through post reporting
- **Payment processing errors**: Display mock payment interfaces with success/failure states for all purchases (tips, ads, followers, verification)
- **Ad conflicts**: Prevent multiple users from purchasing the same position for overlapping time periods
- **Insufficient funds**: Clear messaging when users cannot afford purchases with helpful earning suggestions
- **Empty feed states**: Show onboarding content and suggested follows
- **Missing hashtags**: Empty hashtag explorer with guidance to use hashtags in posts
- **Verification status**: Display verified badges consistently across all user displays
- **Follower scaling**: Cap maximum like multiplier to prevent abuse while rewarding high follower counts
- **Offline functionality**: Cache recent content and show offline indicator
- **Empty notifications**: Show helpful empty state when user has no notifications yet
- **Cookie consent persistence**: Store cookie preference and only show popup once per user
- **Notification overflow**: Limit notification display and provide scroll for long lists
- **Large file uploads**: Handle avatar and banner image uploads with size validation and compression
- **Empty search results**: Show helpful empty states when no users or posts match search query
- **Duplicate reports**: Allow multiple reports on same post for tracking severity
- **Video evidence size limits**: Cap video uploads at 50MB with clear error messaging for oversized files
- **Missing video evidence**: Allow reports without video proof, making video attachment optional
- **Report review overflow**: Paginate or scroll large lists of reports in admin panel
- **Deleted post reports**: Automatically remove all reports when a post is deleted by admin or author
- **Favorite users slots full**: Show clear messaging when all 5 slots are occupied, display available slots count
- **Expired rentals**: Automatically remove expired favorite user slots from display
- **Withdrawal limits**: Prevent withdrawals exceeding available balance, require minimum payment method setup
- **Footer page conflicts**: Allow unlimited custom footer pages, organize by category
- **Ad editing restrictions**: Only allow editing of scheduled ads that haven't started yet, prevent editing live ads
- **Ad deletion policy**: Provide 85% refund for deleted scheduled ads, prevent deletion of live ads
- **Multiple scheduled ads**: Display clear status badges (LIVE vs SCHEDULED) for all user ads
- **Refund calculations**: Automatically calculate and process 85% refunds when ads are cancelled

## Design Direction
The design should feel modern and engaging like Twitter but with strong political theming - bold party colors, patriotic elements, and clear visual hierarchy that makes political alignment immediately apparent while remaining accessible to all viewpoints.

## Color Selection
Triadic color scheme using traditional American political colors to create clear party distinction while maintaining visual harmony.

- **Primary Color**: Deep Blue (oklch(0.45 0.15 250)) - Democratic primary, conveys trust and stability
- **Secondary Colors**: 
  - Bold Red (oklch(0.55 0.20 25)) - Republican primary, represents strength and tradition
  - Neutral Gray (oklch(0.65 0.02 260)) - Independent option, shows balance and moderation
- **Accent Color**: Gold (oklch(0.75 0.15 85)) - Tipping and premium features, suggests value and reward
- **Foreground/Background Pairings**: 
  - Background White (oklch(0.98 0.01 260)): Dark text (oklch(0.15 0.02 260)) - Ratio 16.2:1 ✓
  - Primary Blue (oklch(0.45 0.15 250)): White text (oklch(0.98 0.01 260)) - Ratio 8.9:1 ✓  
  - Primary Red (oklch(0.55 0.20 25)): White text (oklch(0.98 0.01 260)) - Ratio 6.1:1 ✓
  - Accent Gold (oklch(0.75 0.15 85)): Dark text (oklch(0.15 0.02 260)) - Ratio 7.8:1 ✓

## Font Selection
Professional yet approachable typography that conveys authority and trustworthiness while remaining highly legible for political discourse.

- **Typographic Hierarchy**: 
  - H1 (Platform Title): Inter Bold/32px/tight letter spacing
  - H2 (Section Headers): Inter Semibold/24px/normal spacing  
  - H3 (Post Authors): Inter Medium/18px/normal spacing
  - Body (Post Content): Inter Regular/16px/relaxed line height
  - Caption (Metadata): Inter Regular/14px/muted color

## Animations
Subtle and purposeful animations that enhance usability without distraction - political discourse should feel serious and trustworthy, not gimmicky.

- **Purposeful Meaning**: Smooth transitions between party themes, gentle hover states on interactive elements, satisfying like/tip animations
- **Hierarchy of Movement**: Primary focus on engagement feedback (likes, tips), secondary on navigation transitions, minimal on decorative elements
